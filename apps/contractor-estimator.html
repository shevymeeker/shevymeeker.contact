Offline-first structure

Normalized Dexie.js schema: clients, projects, lineItems; cache-first service worker for UI assets; jsPDF for proposal exports.

  • projects table references clientId; lineItems reference projectId.
  • Indexed fields on client name and project status for fast search.
  • Service worker uses cache-first for shell, network fallback for new data.
  • Sales CTA stays outside the iframe so you can explain the offer.
