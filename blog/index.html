Offline-first playbook

How every PWA in this portfolio is structured for installability, offline use, and GitHub Pages deployment.

  • App shell cached via service worker (cache-first) so navigation works offline.
  • Dexie.js as a typed wrapper for IndexedDB; separate tables per entity to stay normalized.
  • Manifest shipped from the repo so install prompts work on desktop and mobile.
  • jsPDF/react-pdf for on-device PDFs; pulls relational data from Dexie tables.
  • Deployed to GitHub Pages; service worker scope stays at the repo root.
